Critical type settings:

  • Hero line-height: 0.85 — tighter than default, essential for the stacked look
  • Hero letter-spacing: -0.04em — negative tracking makes display text feel intentional
  • Label letter-spacing: 0.2em — wide-spaced provides structural contrast
  • Body line-height: 1.625 (Tailwind leading-relaxed)

4. Page-by-Page Layout

4.1 Home (/)

Navigation

  • Full-width flex row, padding: 2rem 4rem (fluid), justify-between items-start
  • Left: brand logotype — Fraunces semibold italic, 1.5rem
  • Right: 3–4 DM Sans medium links, 0.875rem, gap 2.5rem
  • Hover: coral dot appears left of link + text → #FF4530
  • No border, no background — nav floats over hero

Hero Section

  • Role label row: coral 8px dash + uppercase DM Sans, margin-bottom: 3rem
  • <h1>: two-line stacked Fraunces at clamp(4rem, 10vw, 11rem), line-height: 0.85, letter-spacing: -0.04em
  • Line 1 — medium upright. Line 2 — light italic; accent word in coral with curved SVG underline
  • Below-fold grid (12-col): bio sentence cols 1–5, coral pill CTA right-aligned cols 6–12

Featured Work

  • Section label → 2-column ProjectCard grid → "View all work →" coral text link

4.2 Work Index (/work)

  • Page title: Fraunces medium, 4–6rem
  • Filter tabs: DM Sans small caps — All / Brand / Web / Motion. Active: coral underline
  • 2-column grid desktop, 1-column mobile, gap: 2–3rem
  • Card hover: image scale(1.03), coral accent mark

4.3 Project Detail (/work/[slug])

  • Full-bleed hero image or video
  • Meta row: client · year · category in DM Sans caps, coral dash separators
  • Body: DM Sans light, max-width: 40rem
  • Footer: "Next: Project Name →" — Fraunces italic, coral

4.4 About (/about)

  • Headline: Fraunces medium/italic, 4rem+
  • Portrait photo: asymmetric (right col desktop)
  • Bio: 2–3 paragraphs DM Sans light
  • Disciplines: coral dash + category name

4.5 Contact (/contact)

  • Headline: Fraunces italic — "Let's make something."
  • Form: name, email, message — underline-only borders, coral focus state
  • Submit: coral pill button
  • Below: email as coral text link + social icons

8. Implementation Notes

Stack

  • Next.js 15 (App Router) + TypeScript + Tailwind CSS v4
  • Deploy on Vercel — zero-config, auto-deploys on push to main
  • Icons: lucide-react

Bootstrap commands

npx create-next-app@latest my-portfolio \
  --typescript --tailwind --app --eslint --src-dir=false
cd my-portfolio
npm install lucide-react

Fluid hero type

/* CSS */
font-size: clamp(4rem, 10vw, 11rem);

/* Tailwind 4 */
className="text-[clamp(4rem,10vw,11rem)]"

Fraunces optical size (variable font)

/* Set opsz axis for max optical character at display sizes */
.hero-text-italic  { font-variation-settings: 'opsz' 144, 'wght' 300; font-style: italic; }
.hero-text-upright { font-variation-settings: 'opsz' 144, 'wght' 500; }

GitHub → Vercel connection

  1. Run git init && git add . && git commit -m "init"
  2. Create a repo on GitHub and push
  3. Import the repo in the Vercel dashboard — auto-deploys on every push to main
